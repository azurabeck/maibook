# SubsectionPage

## Como funciona

Componente `SubsectionPage` organizado no padrão de pasta do projeto.

- `index.tsx`: implementação principal.
- `type.ts`: tipos públicos do componente.
- `css.ts`: nomes de classes usados pelo componente, exportados como constante.
- `doc.md`: documentação rápida de uso.

## Exemplo de import

```tsx
import { SubsectionPage } from '@/components/molecules/SubsectionPage'
```
