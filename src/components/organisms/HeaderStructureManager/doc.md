# HeaderStructureManager

Gerencia os modelos de cabeçalho de um projeto.

- Lista os modelos salvos no Firestore.
- Abre um editor visual com quatro estruturas fixas.
- Permite criar, editar e excluir cabeçalhos.
- Mostra uma prévia em tempo real.

## Importação

```tsx
import { HeaderStructureManager } from '@/components/organisms/HeaderStructureManager'

<HeaderStructureManager projectId={projectId} />
```
