# ImagesTab

## Como funciona

Página `ImagesTab` organizada no mesmo padrão dos components.

- `index.tsx`: implementação principal.
- `type.ts`: tipos públicos/locais.
- `css.ts`: nomes de classes usados pela página.
- `doc.md`: documentação rápida.

## Exemplo de import

```tsx
import { ImagesTab } from '@/pages/project/ImagesTab'
```
