# TimelineTab

## Como funciona

Página `TimelineTab` organizada no mesmo padrão dos components.

- `index.tsx`: implementação principal.
- `type.ts`: tipos públicos/locais.
- `css.ts`: nomes de classes usados pela página.
- `doc.md`: documentação rápida.

## Exemplo de import

```tsx
import { TimelineTab } from '@/pages/project/TimelineTab'
```
